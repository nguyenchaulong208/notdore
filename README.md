# 📘 README — NotDore

NotDore là nền tảng hỗ trợ kế toán – thuế – nhân sự với các tính năng lưu trữ tài liệu, công cụ tính toán và các tiện ích phục vụ công việc chuyên môn. Dự án hướng đến việc xây dựng một hệ thống đơn giản, dễ dùng, và có khả năng mở rộng trong tương lai.

---

## ✨ Tính năng chính

- **Kho tài liệu**  
  Lưu trữ và quản lý các văn bản, biểu mẫu, tài liệu phục vụ công việc.

- **Công cụ làm việc**  
  Các tiện ích hỗ trợ tính toán và xử lý nghiệp vụ.

- **Giao diện hiện đại**  
  Sử dụng template DevBook để tạo trải nghiệm trực quan và dễ sử dụng.

- **Hệ thống quản lý nội dung**  
  Cho phép thêm, sửa, xóa và quản lý tài nguyên một cách linh hoạt.

---

## 🧱 Công nghệ sử dụng

- **Frontend**: HTML, Bootstrap 5, DevBook Template  
- **Backend**: FastAPI  
- **Database**: PostgreSQL  
- **Triển khai**: Vercel (Frontend), Backend tùy môi trường  
- **Lưu trữ file**: Cloud storage tùy chọn  

---

### 3. Chạy frontend

Frontend là static, có thể mở trực tiếp hoặc deploy lên Vercel.

---

## 📌 Mục tiêu phát triển

- Mở rộng kho tài liệu  
- Tích hợp thêm công cụ nghiệp vụ  
- Cải thiện giao diện và trải nghiệm người dùng  
- Tối ưu hệ thống quản lý nội dung  
- Hỗ trợ phân tích tài liệu nâng cao trong tương lai
