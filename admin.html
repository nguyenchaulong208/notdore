Roadmap crawl văn bản
v2 (hiện tại) — Nhập tay (nhiều dòng) + Import Excel → sinh SQL script
v3 (kế hoạch) — Crawl từ nguồn (vbpl.vn, thuvienphapluat.vn…) → auto populate form → sinh SQL
v4 — Crawl tự động theo lịch, dedup theo số hiệu, push trực tiếp vào Supabase
